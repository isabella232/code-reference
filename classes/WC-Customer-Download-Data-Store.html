class-wc-customer-download-data-store.php : 280

Get array of download ids by specified args.

public get_downloads( [ $args : array = array() ] ) : array
Parameters
$args : array = array()

Arguments to filter downloads. $args['return'] accepts the following values: 'objects' (default), 'ids' or a comma separeted list of fields (for example: 'order_id,user_id,user_email').

Return values
array —

Can be an array of permission_ids, an array of WC_Customer_Download objects or an array of arrays containing specified fields depending on the value of $args['return'].
